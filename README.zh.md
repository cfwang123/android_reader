# 安卓文本阅读器

轻量阅读器：书架、TXT / EPUB / MOBI / PDF、朗读与导出语音、本地识图、中英文界面。电子书共用一套阅读页；PDF 单独一套。

[English](README.md)

## 功能概览

| 模块 | 能力 |
|------|------|
| **书架** | 导入 TXT/PDF/EPUB/MOBI（及 AZW 等）或整个文件夹；一级书架；绑定外部目录；多选移动/删除；搜索；备份/恢复 |
| **电子书** | TXT / EPUB / MOBI：大书先出第一屏，再后台继续加载 |
| **PDF** | 连续/单页、缩放、切边、右侧快速滚动、目录、书内链接、朗读、扫描页识字、按页导出语音 |
| **朗读 / 导出** | 系统朗读、当前句高亮、锁屏可继续听、通知栏控制；导出 MP3 / M4A / WAV |
| **识图** | 相册或拍照，手机本地识别，不上传；扫描版 PDF 可识字 |
| **其它** | 竖屏/横屏/自动；全屏；界面中英文；常亮与空闲息屏 |

### 书架

- 导入文件或文件夹；绑定文件夹（只浏览，不复制）
- 多选：全选 / 移动 / 删除（不删磁盘上的原文件）
- 阅读历史；进度与书签会记住
- 设置里可 **备份 / 导入** 书架与进度（仅本机，不上传）
- 设置 → **检查更新**（从发布页下载安装新版本）
- ⋮ 菜单：文本转语音、识图

### 电子书阅读（TXT / EPUB / MOBI）

- **打开**：大书先显示前面部分，再后台加载，顶栏有进度
- **主题**：默认 / 纯白 / 护眼绿 / 淡蓝 / 淡紫 / 羊皮纸 / 夜间 / **自定义背景色**
- **字体**：默认 / 无衬线 / 衬线 / 等宽；可 **安装自己的字体文件**（长按卸载）；字号、行距、段距、字距
- **TXT**：自动或手动选编码；简繁转换；一行一段
- **EPUB / MOBI 样式**（常见能力）：
  - 粗体 / 斜体 / 下划线 / 字色 / 背景色
  - 整段大图与行内小图
  - 书内跳转与外链（外链会确认）
  - 点图进入图库（缩放、左右滑、侧边点按翻图）
- **目录 / 书签 / 跳转**；顶栏进度、电量、时钟
- **书内搜索**：边搜边出结果，点一下跳转
- **手势**：两侧点按翻页、滑动滚动；返回可只停朗读、不退出书
- **朗读**：当前句高亮；锁屏/后台可继续听（见下）；可设休眠
- **导出语音**：默认全书，可选起止行；支持 MP3 / M4A / WAV 与码率

### PDF 阅读

- 连续滚动或单页；双指缩放；切边（可自动裁边）
- **快速滚动**：连续模式下右侧进度条（拖动定位；滚动时出现，停约 1 秒后隐藏）
- **目录**：打开后后台准备，点目录跳转快
- **书内链接**：跳页；外链会确认；可返回 / 前进
- **文字**：有文字的页可选中复制；扫描图可 **识别扫描版 PDF 文字**（可选页范围、可取消）
- **朗读 / 导出**：有文字或已识别后可用；导出按页码范围
- **手势**：两侧点按上/下翻页；中间点开菜单
- 菜单里上一页/下一页不会关掉底部菜单

### 朗读与导出

| 能力 | 说明 |
|------|------|
| 系统朗读 | 选引擎、语言、发音人、语速；当前句高亮；休眠定时 |
| 听感 | 提前准备后面几句，减少句与句之间的卡顿 |
| 锁屏续听 | 见下方说明 |
| 导出 | 分段生成再合并；优先 MP3，不行则 M4A / WAV |
| 码率 | 32–192 kbps（MP3/M4A） |
| 独立页 | 书架 ⋮ → 文本转语音：粘贴文字播放或导出 |

### 锁屏 / 后台还能听

目标：锁屏、切到别的应用后仍能接着读，并能用通知栏、锁屏按钮控制。

**应用内会做的事（简要）**

- 朗读时显示通知、尽量不让系统休眠掐断
- 锁屏上可暂停/继续/上一句/下一句
- 灭屏时尽量忽略系统误发的「暂停」
- 句与句之间尽量接上，卡住时会自动重试

锁屏时 **不会因为你离开阅读页就停读**；进度仍会保存。

**若你的手机仍会杀掉朗读**

1. 第一次点朗读时 **允许通知**
2. 系统设置 → 应用 → 本应用 → **省电 / 后台耗电 → 无限制**（小米、华为等）
3. 多任务界面 **锁定** 本应用
4. 确认已装系统朗读引擎与中文语音包

**自测**

1. 打开书 → 朗读 → 通知里应显示「正在朗读」
2. 锁屏听 **1 分钟以上** 是否连续
3. 通知栏 / 锁屏：暂停、继续、上一句、下一句

### 识图

- 书架 ⋮ → 识图
- 在手机本地识别，不上传图片
- 可在图上拖选区域，全文可滑动查看

## 快速开始

```powershell
cd reader
.\gradlew.bat assembleDebug
node build.js run          # 编译 + 安装到手机 + 启动
node build.js devices
node build.js release      # 正式包
node build.js apk          # 正式包并复制到 release/ 目录
```

安装包路径：

```
app/build/outputs/apk/debug/app-debug.apk
app/build/outputs/apk/release/reader{version}.apk
release/reader{version}.apk
```

变更记录见 [CHANGELOG.md](CHANGELOG.md)。

## 常见问题

### 朗读没声音

安装系统朗读引擎与中文语音包；检查音量。

### 锁屏后很快停

见 **锁屏 / 后台还能听**：允许通知；省电设为无限制；多任务里锁定应用。也可能是别的应用抢走了声音。

### TXT 乱码

可自动识别常见编码，或在阅读页手动选编码。

### EPUB / MOBI 打开慢或样式不全

大书会先显示前面再继续加载。只支持常见样式（粗斜体、颜色、图片、链接等），复杂排版、表格可能不完整。加密书打不开。

### PDF 是扫描图，没有字

用菜单 **识别扫描版 PDF 文字**；朗读/导出需要识别后的文字。

### PDF 滚动白页或页面压扁

尽量用最新版；连续滚动时先预览、停稳后再变清晰。仍异常可重新打开该书。

### 不能导出 MP3

部分模拟器或非 64 位设备可能没有 MP3，会自动改用 M4A。

### 书内链接点不了

PDF 需要自带链接；EPUB 需要书里有链接。纯文字目录请用「目录」面板。
