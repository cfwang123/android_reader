# 安卓文本阅读器

轻量阅读器：书架、TXT / EPUB / MOBI / PDF、朗读与导出语音、本地识图、中英文界面。电子书共用一套阅读页；PDF 单独一套。

[English](README.md)

## 功能概览

| 模块 | 能力 |
|------|------|
| **书架** | 导入 TXT/PDF/EPUB/MOBI（及 AZW 等）或整个文件夹；一级书架；绑定外部目录；多选移动/删除；搜索；备份/恢复；阅读历史；长按清除本地记录 |
| **电子书** | TXT / EPUB / MOBI：大书先出第一屏，再**后台预载全书** |
| **MOBI** | 正文阅读；**浏览模式**：正文 / 单图 / 连续图；无文字纯图书自动进图片模式；改进 UTF-8 中文 MOBI |
| **PDF** | 连续/单页、缩放、按文件切边、右侧快速滚动、目录、书内链接、朗读、长页分块 OCR（局部页会重识）、按页导出语音 |
| **朗读 / 导出** | 系统朗读、当前句高亮、锁屏可继续听、通知栏控制；导出 MP3 / M4A / WAV |
| **识图** | 相册或拍照，手机本地识别，不上传；扫描版 PDF 可识字（长图分块） |
| **其它** | 界面颜色主题（16 套）；**竖屏/横屏锁定**（切换时菜单可保持）；全屏；界面中英文；常亮与空闲熄屏；音量键翻页 |

### 书架

- 导入文件或文件夹；绑定文件夹（只浏览，不复制）
- 列表：紧凑行、多选框在右侧
- 多选：全选 / 移动 / 删除（不删磁盘上的原文件）
- **阅读历史**，格式扩展名正确（TXT / PDF / EPUB / MOBI…）
- 进度与书签会记住
- 长按 → **详情**（名称、路径、格式、大小、进度、上次阅读等）
- 长按 → **清除记录**：进度、书签、EPUB/MOBI 章节缓存、PDF 切边/OCR 等（不删书架条目与源文件）
- 退出阅读回到书架时 **刷新进度与上次阅读**
- 设置里可 **备份 / 导入** 书架与进度（仅本机，不上传）
- 设置 → **检查更新**（从发布页下载安装新版本）
- 设置 → **外观 → 颜色主题**（16 套界面皮肤；与阅读页纸张配色独立）
- ⋮ 菜单：文本转语音、识图

### 电子书阅读（TXT / EPUB / MOBI）

- **打开**：大书先显示前面部分，再**后台预载**（EPUB/MOBI）；恢复进度时不闪首页
- **进度**：EPUB/MOBI 显示 **第 n/m 章 + 章内 %**；TXT 为百分比
- **目录**：自动识别章节标题；若无目录或不准，可用 **自定义目录扫描**（通配符 `*`、`x`、`xxx`、`xxxx`）— 如 `第x章 *`、`卷第x *`、`Chapter x`；按书记忆
- **阅读样式**：背景纹理 / 纯色 / 导入图；字色预设 + 自定义 HSV；字号行距；可安装字体（长按卸载）；默认用系统字体（不内嵌商业字体）
- **TXT**：自动或手动选编码；简繁转换
- **EPUB / MOBI 样式**（常见能力）：
  - 粗体 / 斜体 / 下划线 / 字色 / 背景色
  - 整段大图与行内小图
  - 书内跳转与外链（外链会确认）
  - 长按图片进入图库（缩放、左右滑、侧边点按翻图）
- **MOBI 浏览模式**（风格 → 浏览模式）：**正文**（常规阅读）；**单图**（一次一张、双指缩放、侧点/滑动翻页）；**连续图**（竖向图片流）。无正文纯图 MOBI 自动进图片模式；图片模式下进度为 **第 n/m 张**。**竖屏 ↔ 横屏** 时连续图保持缩放与水平平移比例
- **文本选择**：长按选字；两端 **拖动手柄** 调整范围；拖手柄到上下边缘可自动滚屏扩选；复制 / 从此朗读
- **目录 / 书签 / 跳转**；底栏电量、时钟、进度；打开目录时滚到当前章节；竖向列表滚动不会误触横滑切换（目录 ↔ 书签）
- **书内搜索**：边搜边出结果，点一下跳转
- **手势**：两侧点按翻页、滑动滚动；**音量键翻页**（默认开）；**左右贴边 10px** 以 **0.5sp** 步进调字号；返回可只停朗读、不退出书
- **朗读**：当前句高亮；锁屏/后台可继续听（见下）；可设休眠
- **导出语音**：默认全书，可选起止行；支持 MP3 / M4A / WAV 与码率；合成时有进度对话框

### PDF 阅读

- 连续滚动或单页；双指缩放；**切边按文件独立保存**（可自动裁边；奇偶对称可按书开）
- **进度 %**：按滚动位置 / 全书内容总高度（长页内滚动也会变）
- **快速滚动**：连续模式下右侧进度条（拖动定位；滚动时出现，停约 1 秒后隐藏）
- **目录**：打开后后台准备，点目录跳转快
- **书内链接**：跳页；外链会确认；可返回 / 前进
- **文字**：有文字的页可选中复制，**手柄** 可调整选区；扫描图可 **识别扫描版 PDF 文字**（长页分块 OCR；**仅识别到局部的竖长页会分块重识**；可选页范围、可取消）
- **朗读 / 导出**：有文字或已识别后可用；连续模式下高亮随滚动跟随并避开 TTS 栏；导出按页码范围
- **手势**：两侧点按上/下翻页；中间点开菜单
- 菜单里上一页/下一页不会关掉底部菜单；**横竖屏切换时菜单可保持，图标仍为 2×4 布局**；**连续模式**旋转时保持缩放与水平平移比例

### 朗读与导出

| 能力 | 说明 |
|------|------|
| 系统朗读 | 选引擎、语言、发音人、语速；当前句高亮；休眠定时 |
| 听感 | 提前准备后面几句，减少句与句之间的卡顿 |
| 锁屏续听 | 见下方说明 |
| 导出 | 分段生成再合并；优先 MP3，不行则 M4A / WAV；实时进度 |
| 码率 | 32–192 kbps（MP3/M4A） |
| 独立页 | 书架 ⋮ → 文本转语音：粘贴文字播放或导出 |

### 锁屏 / 后台还能听

目标：锁屏、切到别的应用后仍能接着读，并能用通知栏、锁屏按钮控制。

**应用内会做的事（简要）**

- 朗读时显示通知、尽量不让系统休眠掐断
- 锁屏上可暂停/继续/上一句/下一句
- 灭屏时尽量忽略系统误发的「暂停」
- 句与句之间尽量接上，卡住时会自动重试

锁屏时 **不会因为你离开阅读页就停读**；进度仍会保存。

**若你的手机仍会杀掉朗读**

1. 第一次点朗读时 **允许通知**
2. 系统设置 → 应用 → 本应用 → **省电 / 后台耗电 → 无限制**（小米、华为等）
3. 多任务界面 **锁定** 本应用
4. 确认已装系统朗读引擎与中文语音包

**自测**

1. 打开书 → 朗读 → 通知里应显示「正在朗读」
2. 锁屏听 **1 分钟以上** 是否连续
3. 通知栏 / 锁屏：暂停、继续、上一句、下一句

### 识图

- 书架 ⋮ → 识图
- 在手机本地识别，不上传图片
- 可在图上拖选区域，全文可滑动查看

## 快速开始

```powershell
cd reader
.\gradlew.bat assembleDebug
node build.js run          # 编译 + 安装到手机 + 启动
node build.js devices
node build.js release      # 正式包
node build.js apk          # 正式包并复制到 release/ 目录
```

安装包路径：

```
app/build/outputs/apk/debug/app-debug.apk
app/build/outputs/apk/release/reader{version}.apk
release/reader{version}.apk
```

## 常见问题

### 朗读没声音

安装系统朗读引擎与中文语音包；检查音量。

### 锁屏后很快停

见 **锁屏 / 后台还能听**：允许通知；省电设为无限制；多任务里锁定应用。也可能是别的应用抢走了声音。

### TXT 乱码

可自动识别常见编码，或在阅读页手动选编码。

### EPUB / MOBI 打开慢或样式不全

大书会先显示前面再**后台预载全书**。只支持常见样式（粗斜体、颜色、图片、链接等），复杂排版、表格可能不完整。加密书打不开。

### MOBI 中文乱码

新版已修复部分 UTF-8 中文 MOBI（如编码误标、PalmDOC 解压后 HTML 损伤）。若仍乱码，可重新打开或长按清除本地记录（旧解析缓存）。

### 没有目录 / 章节不对

打开 **目录 → 自定义目录扫描**，选预设或输入通配符（如 `第x章 *`、`001.` 样式 `xxx. *` 等），点 **应用**。该书会记住模式。

### MOBI 是纯图 / 漫画

用 **风格 → 浏览模式 → 单图** 或 **连续图**，或打开无正文的纯图 MOBI（会自动进图片模式）。进度为第 n/m 张。

### PDF 是扫描图，没有字

用菜单 **识别扫描版 PDF 文字**；竖长页会分块识别。若以前只识到页面上部，可再次 OCR（不勾选跳过已有）——局部页会分块重识。朗读/导出需要识别后的文字。

### 选中与复制文字

长按选字（英文会先扩成整词），拖动可扩展选区，松手后用 **两端手柄** 微调；拖手柄到屏幕上下边缘可滚屏扩选。PDF 需有提取文字或已 OCR。

### PDF 滚动白页或页面压扁

尽量用最新版；连续滚动时先预览、停稳后再变清晰。仍异常可重新打开该书。

### 不能导出 MP3

部分模拟器或非 64 位设备可能没有 MP3，会自动改用 M4A。

### 书内链接点不了

PDF 需要自带链接；EPUB 需要书里有链接。纯文字目录请用「目录」面板。
